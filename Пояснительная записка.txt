Битва ботов
Автор - Хлопина Софья
Боты играют в игру - каждый выбирает натуральное число, у кого наименьшее и неповторяющееся - тот победил. Результаты игр заносятся в базу данных. В проекте можно выбирать ботов, которые будут играть, смотреть статистику их побед, сбрасывать данные, менять дизайн, смотреть историю всех игр. Большинство ботов берут значения из уже прошедших игр, каждый выбирает своим способом.
Каждый бот - отдельный класс: Red (чем больше число, тем меньше шанс, что бот его выберет), Blue (выбирает самое редкое из чисел в базе данных), Yellow (выбирает самое редкое из чисел, которые он уже выбирал), Green (выбирает самое редкое из выигрывавших чисел), Orange (выбирает самое редкое из выигрывавших чисел, которые не он выбирал), Purple (выбирает самое редкое из чисел в базе данных и умножает его на случайное число, большее 0 и меньшее 1), Pink (чем чаще число выигрывает - тем чаще его выбирает бот), Gray (перебирает выигрывавшие числа, выбирает то, когда случайное число, большее 0 и меньшее 1 меньше 0.1, как только прошло уже 100 чисел - выбирается случайное число), Black (выбирает случайное число от 1 до 10), White (выбирает минимальное из выигрывавших чисел), Brown (выбирает всегда 1), Violet (у него есть коэффициент k, на который умножается последнее выигравшее число, после каждого выигрыша число уменьшается на случайное число, большее 0 и меньшее 1). Если бот берёт число из базы данных, а там нет чисел, то он берёт случайное число от 1 до 5. У каждого бота число возвращает функция Run().
Есть класс Game, реализующий саму игру и взаимодействие ботов.
Используются библиотеки random, sys, PyQt5, sqlite3.
По результатам 500 игр пока лидирует Brown, выдающий постоянно 1. Меньше всего побед у Yelow, который выбирает самое редкое из чисел, которые уже выбирал.
